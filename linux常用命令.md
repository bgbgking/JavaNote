# 常用命令

## 目录

cd: 改变目录

cd .. 返回上一级目录 

pwd 显示当前路径

ls 查看目录中的文件 

mkdir dir1 创建一个叫做 'dir1' 的目录' 

rm -rf dir1 删除一个叫做 'dir1' 的目录并同时删除其内容 

mv dir1 new_dir 重命名/移动 一个目录 



## 文件

touch 在当前目录创建一个文件

rm -f file1 删除一个叫做 'file1' 的文件' 

mv 1.html src 1.html 移动文件（src是目标文件夹）

## 系统

clear 清屏

help 帮助

exit 退出

history 查看命令历史

“#”  井号表示注释
